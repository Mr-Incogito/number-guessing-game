# Number Guessing Game

## Overview
A command-line number guessing game built in Python. The program generates a random number and prompts the user to guess until the correct number is entered.

## Features
- Random number generation
- Input validation
- Loop-based gameplay
- User feedback for high/low guesses

## Technologies Used
- Python 3

## How to Run
1. Clone the repository:
   git clone https://github.com/jbeale-dev/number-guessing-game.git

2. Navigate to the folder:
   cd number-guessing-game

3. Run the program:
   python guessing_game.py

## Learning Objectives
This project demonstrates:
- Conditional statements
- Loops
- Functions
- Basic algorithm logic
